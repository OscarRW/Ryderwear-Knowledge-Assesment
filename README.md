# Ryderwear Gym — Team Knowledge Assessment

A self-contained, single-page knowledge assessment for Ryderwear Gym team members.

## Deploy to Railway

1. Push this repo to GitHub.
2. In Railway, create a new project from this GitHub repo.
3. Railway auto-detects Node, runs `npm install`, and starts the app with `npm start`.
4. A public domain is generated under **Settings → Networking → Generate Domain**.

The app serves `index.html` as a static site on the port Railway provides via `$PORT`.

## Run locally

```
npm install
npm start
```

Then open http://localhost:3000.
